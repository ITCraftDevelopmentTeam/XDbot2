# 指令帮助 —— calc

## 描述：
该指令用于计算数学表达式。

### 权限：
- everyone

## 用法

### `<calc <表达式>>`

- 说明：该用法用于执行数学表达式的计算。
- 示例：`/calc 2 + 3`

### `<calc <表达式1> - <表达式2>>`

- 说明：该用法用于执行两个数学表达式的减法运算。
- 示例：`/calc 6 - 3`

### `<calc <表达式1> * <表达式2>>`

- 说明：该用法用于执行两个数学表达式的乘法运算。
- 示例：`/calc 4 * 5`

### `<calc <表达式1> / <表达式2>>`

- 说明：该用法用于执行两个数学表达式的除法运算。
- 示例：`/calc 8 / 2`
