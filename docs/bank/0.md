# 指令帮助 —— bank

## 描述：
XDbot2银行指令，用于贷款、查询贷款信息以及还贷等操作。

### 权限：
- `everyone`

## 用法

### `bank lend <money>`

- 说明：向银行贷款指定金额。
- 示例：`/bank lend 1000`

### `bank view`

- 说明：查看当前需要还贷的金额。
- 示例：`/bank view`

### `bank repay <ID>`

- 说明：根据贷款ID还贷指定金额。
- 示例：`/bank repay 123456`
