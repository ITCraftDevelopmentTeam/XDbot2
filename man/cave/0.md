# 指令帮助 —— cave

## 描述：
该指令用于随机、投稿或查询回声洞。

### 权限：
- `everyone`

## 用法

### `cave`

- 说明：随机获取一条回声洞。
- 示例：`/cave`

### `cave-a <内容>`

- 说明：投稿一条回声洞。
- 示例：`/cave-a Hello, world!`

### `cave-s`

- 说明：查看回声洞状态。
- 示例：`/cave-s`

### `cave-c <ID> <内容...>`

- 说明：添加评论。
- 示例：`/cave-c 12345 This is a comment.`
