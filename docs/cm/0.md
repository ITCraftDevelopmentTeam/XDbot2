# 指令帮助 —— cm

## 描述：
该指令允许用户在群内禁用或启用命令。

### 权限：
- `superuser`
- `group_admin`
- `group_owner`
- `group`

## 用法

### `cm block <命令>`

- 说明：禁用指定的命令。
- 示例：`/cm block invite`

### `cm unblock <命令>`

- 说明：启用指定的命令。
- 示例：`/cm unblock invite`

### `cm list`

- 说明：查看被禁用的命令。
- 示例：`/cm list`
