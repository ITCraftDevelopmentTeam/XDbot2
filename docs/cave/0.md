# cave(0)

## 名称

cave - 回声洞

## 用法

1. cave
2. cave add <内容>
3. cave-g <回声洞ID>

## 描述

致敬 PCL内群 罗伯特 的回声洞功能

用户可以自己投稿内容，或抽取其他用户投稿的内容

## 示例

### cave

随机抽取一条回声洞

### cave add 114514

将 114514 添加到回声洞

我们不建议您这么做，因为经常回声洞投稿垃圾内容这样可能会被超级用户封禁（见su(1)）

关于回声洞投稿守则，见cave(1)

### cave-g <回声洞ID>

显示ID为 <回声洞ID> 的回声洞
