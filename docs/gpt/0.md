# 指令帮助 —— 命令名

## 描述：
该指令允许使用XDbot2GPT生成文本。

### 权限：
- `everyone`

## 用法

### `gpt info`

- 说明：获取XDbot2GPT信息
- 示例：`gpt info`

### `gpt reset [模板]`

- 说明：重置XDbot2GPT
- 示例：`gpt reset 模板`

### `gpt switch {global|group|private}`

- 说明：切换XDbot2GPT的范围
- 示例：`gpt switch global`

### `gpt token buy <数量>`

- 说明：购买XDbot2GPT的访问令牌
- 示例：`gpt token buy 10`

### `gpt <内容>`

- 说明：使用XDbot2GPT生成文本
- 示例：`gpt 请写一封推荐信给甲公司`
