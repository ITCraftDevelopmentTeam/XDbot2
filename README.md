## 简介

XDbot2 是一个简单的QQ功能型机器人，基于 Nonebot2 + Onebot V11 编写

相比于[XDbot](https://github.com/Moonlark-Dev/XDbot)，XDbot2拥有这些特性

- 更稳定：更强的错误处理和上报系统
- 便于管理：强大的超管系统，轻松管理XDbot
- 易于使用：更详细的指令文档和更易用的指令
- 功能更丰富：在XDbot的基础上增加了更多功能
