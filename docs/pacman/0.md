# 指令帮助 —— pacman

## 描述：
在 archlinux.org/packages 上搜索指定的包名。

### 权限：
- everyone

## 用法

### `<用法1>`

- 说明：搜索指定包名并返回搜索结果。
- 示例：`/pacman firefox`

### `<用法二>`

- 说明：搜索指定包名并返回详细信息。
- 示例：`/pacman -i firefox`

### `<用法三>`

- 说明：搜索包含指定关键词的包。
- 示例：`/pacman -s text editor`
